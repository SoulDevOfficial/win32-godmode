# win32-godmode
**Description:** Command to setup the Godmode application on your windows device!

# How to Use
Simply right click anywhere on your home page -> new -> file than name it `GodMode.{ED7BA470-8E54-465E-825C-99712043E01C}`. The file icon should then turn into something that looks like the MS control pannel however this one with provide full access to all windows settings including ones that are hidden.
